## 一、搜索方案评估

### 1.1 标签搜索 (Tag-based Search)

#### 实现方式
利用现有 `broadcast_data.json` 的数据结构：
```json
{
  "scenarios": {
    "三超行李广播": [...],
    "登机广播": [...],
    "催促广播": [...]
  }
}
```

- 一级分类：scenarios 键名
- 二级标签：每个广播类型的 `广播类型` 字段
- 点击卡片进入对应场景列表

#### 优势
- 零额外开发成本
- 符合机场工作人员心智模型
- 离线可用，无网络依赖
- 触控友好（大卡片设计）

#### 实现复杂度
**P0 - 极低** (0.5天)

### 1.2 关键词召回 (Keyword Matching)

#### 实现方式
纯前端实现，无需服务器：

```javascript
function searchBroadcasts(query) {
  const results = [];
  const q = query.toLowerCase();

  for (const [scenario, items] of Object.entries(data.scenarios)) {
    items.forEach(item => {
      // 搜索范围：广播类型名 + 中文/英文/粤语内容
      const match =
        item['广播类型'].includes(query) ||
        item['中文'].includes(query) ||
        item['英文'].toLowerCase().includes(q) ||
        item['粤语'].includes(query);

      if (match) {
        results.push({
          scenario,
          type: item['广播类型'],
          content: item
        });
      }
    });
  }
  return results;
}
```

#### 性能指标
- 数据量：约50-100条广播词
- 搜索响应：<50ms
- 内存占用：<1MB

#### 实现复杂度
**P0 - 低** (0.5天)

## 二、模板可填项识别与表单生成方案

### 2.1 变量模式识别

#### 从广播词中提取的变量模式

| 变量类型 | 正则模式 | 示例 | 输入方式 |
|----------|----------|------|----------|
| 航班号 | `/CA\d{4}/gi` | CA1234 | 数字键盘 + CA前缀 |
| 登机口 | `/C?\d{2}/gi` | C15, 21 | 数字键盘 |
| 时间 | `/\d{1,2}:\d{2}/gi` | 15:30 | 时间选择器 |
| 传送带号 | `/\d{1,2}号?传送带/gi` | 45号传送带 | 数字键盘 |
| 占位符 | `/___+/g` | ___ | 文本输入 |
| 混合占位 | `/CAXXXX|CXX/gi` | CAXXXX | 文本输入 |

#### 识别实现

```javascript
function extractVariables(template) {
  const patterns = {
    flightNo: /CA\d{4}|CA{4,5}/gi,
    gate: /C?\d{2}|CXX/gi,
    time: /\d{1,2}:\d{2}/gi,
    carousel: /\d{1,2}号?传送带/gi,
    placeholder: /_{3,}/g
  };

  const variables = [];

  for (const [type, pattern] of Object.entries(patterns)) {
    const matches = template.match(pattern);
    if (matches) {
      matches.forEach(m => {
        if (!variables.find(v => v.value === m)) {
          variables.push({ type, value: m, label: getLabel(type) });
        }
      });
    }
  }

  return variables;
}

function getLabel(type) {
  const labels = {
    flightNo: '航班号',
    gate: '登机口',
    time: '时间',
    carousel: '行李传送带',
    placeholder: '请填写'
  };
  return labels[type] || '未知';
}
```

### 2.3 表单组件设计

#### 移动端输入优化

| 变量类型 | 输入方式 | 默认值 | 验证规则 |
|----------|----------|--------|----------|
| 航班号 | 数字键盘 | 航班API(未来) | `/^CA\d{4}$/i` |
| 登机口 | 数字键盘 | 航班API(未来) | `/^\d{2}$/` |
| 目的地 | 搜索下拉 | 航班API(未来) | 非空 |
| 时间 | 时间选择器 | 当前时间 | HH:MM格式 |

#### 表单验证实现

```javascript
const validators = {
  flightNo: v => /^CA\d{4}$/i.test(v),
  gate: v => /^\d{2}$/.test(v),
  time: v => /^\d{1,2}:\d{2}$/.test(v),
  notEmpty: v => v && v.trim().length > 0
};

function validateForm(formData, schema) {
  const errors = [];

  for (const [field, rule] of Object.entries(schema)) {
    const value = formData[field];
    if (!validators[rule](value)) {
      errors.push({ field, message: `${field}格式错误` });
    }
  }

  return errors;
}
```
